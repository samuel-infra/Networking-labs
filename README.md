# Networking Labs (Cisco Packet Tracer)

## Overview
This repository contains hands-on networking labs focused on fundamental networking concepts using Cisco Packet Tracer.

---

## Topics covered

- Basic network configuration
- IP addressing and static IP setup
- Switch configuration
- VLAN basics
- Subnetting
- Connectivity testing (ping)

---

## Purpose

These labs demonstrate practical understanding of networking fundamentals and troubleshooting in simulated environments.

---

## Tools

- Cisco Packet Tracer
- CLI configuration

---

## Note

Each lab includes a brief explanation and configuration details.
