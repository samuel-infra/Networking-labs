# Basic Routing Between Two LANs

## Objective
Configure routing between two separate local networks using a router.

---

## What I did

- Configured IP addresses on PCs and router interfaces
- Connected two LAN networks via a router
- Enabled routing between networks
- Verified connectivity using ping

---

## Result

- Devices in different networks successfully communicated
- Routing between LANs was fully functional

---

## Key concepts

- IP addressing
- Default gateway
- Basic routing
