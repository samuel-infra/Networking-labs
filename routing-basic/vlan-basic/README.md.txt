# Basic VLAN Configuration

## Objective
Configure VLANs on a switch and assign devices to different networks.

---

## What I did

- Created VLANs on a switch
- Assigned switch ports to specific VLANs
- Configured IP addresses for devices
- Tested communication within VLANs

---

## Result

- Devices within the same VLAN could communicate
- Devices in different VLANs were isolated

---

## Key concepts

- VLAN segmentation
- Access ports
