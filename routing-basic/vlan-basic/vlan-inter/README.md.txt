# Inter-VLAN Communication

## Objective
Enable communication between different VLANs using routing.

---

## What I did

- Configured multiple VLANs
- Set up router-on-a-stick or L3 routing
- Assigned IP addressing per VLAN
- Verified communication between VLANs

---

## Result

- Devices in different VLANs could communicate
- Routing between VLANs worked correctly

---

## Key concepts

- Inter-VLAN routing
- Trunk ports
- Default gateway per VLAN
