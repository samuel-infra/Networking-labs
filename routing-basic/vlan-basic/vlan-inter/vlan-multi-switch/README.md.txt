# VLAN Across Multiple Switches

## Objective
Extend VLANs across multiple switches and enable communication.

---

## What I did

- Configured VLANs on multiple switches
- Set up trunk links between switches
- Allowed VLAN traffic across switches
- Verified connectivity across the network

---

## Result

- VLANs successfully extended across switches
- Devices could communicate across switches within the same VLAN

---

## Key concepts

- Trunking (802.1Q)
- VLAN propagation
- Switch-to-switch communication
